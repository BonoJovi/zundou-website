# zundou.org - Product Portal

A multi-language portal website introducing three amazing products:
- **KakeiBon** - Personal finance management app
- **Promps** - Prompt management platform
- **ai2ia** - AI assistant

## Features

- 🌐 7-language support (English, 日本語, Français, Italiano, Deutsch, Русский, العربية)
- 📱 Responsive design
- ✨ Beautiful purple gradient UI
- 🚀 Instant deployment to Cloudflare Pages

## Deployment

This site is automatically deployed to Cloudflare Pages on every push to the `main` branch.

### Setup

1. Create a Cloudflare Pages project named `zundou-website`
2. Add the following secrets to your GitHub repository:
   - `CLOUDFLARE_API_TOKEN`
   - `CLOUDFLARE_ACCOUNT_ID`
